   - 选择儿童：测试儿童小明

   - 选择剂次：第1剂

4. **等待管理员审核**
   - 预约状态：待审核
   - 管理员在后台审核通过

5. **触发预约后提醒**
   - 定时任务扫描到已通过的预约
   - 提前3天和1天发送提醒

### 验证点

✅ **数据库验证**
```sql
-- 查看预约记录
SELECT 
    a.id,
    c.name AS child_name,
    v.name AS vaccine_name,
    a.appointment_date,
    a.status,
    CASE a.status
        WHEN 0 THEN '待审核'
        WHEN 1 THEN '已通过'
        WHEN 2 THEN '已拒绝'
        WHEN 3 THEN '已完成'
        WHEN 4 THEN '已取消'
    END AS status_text
FROM appointment a
LEFT JOIN child c ON a.child_id = c.id
LEFT JOIN vaccine v ON a.vaccine_id = v.id
WHERE a.child_id = [儿童ID]
ORDER BY a.create_time DESC;
```

✅ **预期结果**
- appointment 表新增记录
- status = 0（待审核）→ 1（已通过）
- 触发预约后提醒（3天前和1天前）
